## 🎨 User Experience

### The Flow

```
User clicks "Enroll Now" / "Buy Now"
           ↓
    [Auth Check]
           ↓
    ┌──────┴──────┐
 Logged In    Not Logged In
    │             │
    ↓             ↓
 Opens Link   Shows Modal
  (Done!)         ↓
              User Logs In
                  ↓
              Modal Closes
                  ↓
              Link Opens
              Automatically
                  ↓
              Enrollment!
```
